Here are the shortcuts for generating files using the NestJS CLI:


# Generate a Module:
nest g mo <module-name>

# Generate a Controller:
nest g co <controller-name>

# Generate a Service:
nest g s <service-name>

# Generate a Middleware:
nest g mi <middleware-name>

# Generate a Pipe:
nest g pi <pipe-name>

# Generate a Guard:
nest g gu <guard-name>

# Generate an Interceptor:
nest g in <interceptor-name>

# Generate a Gateway (for WebSockets):
nest g ga <gateway-name>

# Generate a Resolver (for GraphQL):
nest g r <resolver-name>
